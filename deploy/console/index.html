Dell 终端镜像制作教程
1

准备 Windows 安装源

下载 Windows 11 24H2 ISO (企业版推荐)。挂载ISO到 D:\win11_iso。准备一台Dell OptiPlex终端（3050/3060/3070/3080/3090/5000/7000均可）。

2

创建 WinPE 基础镜像

使用 Windows ADK 创建 WinPE：

# 安装 Windows ADK + WinPE 插件
# 以管理员运行 Deployment and Imaging Tools Environment

copype amd64 C:\WinPE_HD

# 挂载 WinPE 启动镜像
mount C:\WinPE_HD\media\sources\boot.wim C:\WinPE_mount
3

注入 HyperDisk 组件

将编译好的客户端组件注入WinPE：

# 注入 bootdiag.exe (7项启动诊断)
copy /Y build\bin\Release\bootdiag.exe C:\WinPE_mount\HyperDisk\

# 注入 boot_replay_analyzer.exe (IO录制分析)
copy /Y build\bin\Release\boot_replay_analyzer.exe C:\WinPE_mount\HyperDisk\

# 创建 boot.meta 配置
echo [meta] > C:\WinPE_mount\HyperDisk\boot.meta
echo version=1 >> C:\WinPE_mount\HyperDisk\boot.meta
echo image_id=1 >> C:\WinPE_mount\HyperDisk\boot.meta
echo block_size=4096 >> C:\WinPE_mount\HyperDisk\boot.meta
echo [servers] >> C:\WinPE_mount\HyperDisk\boot.meta
echo primary=10.10.200.10:9090 >> C:\WinPE_mount\HyperDisk\boot.meta
4

添加 Dell 驱动到 WinPE

从 Dell 支持站点 下载对应机型的 WinPE 驱动包（网卡+存储）：

# 注入Dell网卡驱动 (PXE启动必需)
dism /image:C:\WinPE_mount /Add-Driver /Driver:D:\dell_drivers\network /Recurse

# 注入Dell存储驱动 (NVMe必需)
dism /image:C:\WinPE_mount /Add-Driver /Driver:D:\dell_drivers\storage /Recurse

# 常见Dell OptiPlex网卡驱动:
#   Intel I219-V → e1r653x.inf
#   Intel I225-V → e2f683x.inf
注意: 不同Dell机型的驱动不同，务必下载对应型号的WinPE驱动包
5

配置 WinPE 启动脚本

编辑 startnet.cmd 添加HyperDisk启动流程：

echo %TIME% Starting HyperDisk X Boot Agent >> X:\HyperDisk\boot.log

# 运行bootdiag 7项诊断
X:\HyperDisk\bootdiag.exe --config X:\HyperDisk\boot.meta
if %ERRORLEVEL% NEQ 0 (
    echo BOOTDIAG FAILED - Halting >> X:\HyperDisk\boot.log
    pause
    goto :EOF
)

# 启动Boot Replay录制
start /B X:\HyperDisk\boot_replay_analyzer.exe --record

# 连接ImageServer获取块数据
# (由BootAgent自动完成)
6

提交 WinPE 并上传到服务器

# 卸载并提交
dism /unmount-wim /mountdir:C:\WinPE_mount /commit

# 生成WIM文件 (约200-400MB)
# 上传到HyperDisk服务器
scp C:\WinPE_HD\media\sources\boot.wim \
    debian@10.10.300.10:/opt/hyperdisk/data/images/winpe_dell_optiplex.wim

# 通过API注册镜像
curl -X POST http://10.10.300.10/api/v1/images \
  -H "Content-Type: application/json" \
  -d '{"name":"winpe-dell-optiplex-alpha","total_size":419430400,"block_count":102400}'
7

制作完整 Windows 镜像 (可选)

如需完整Windows桌面（非WinPE），使用DISM捕获已安装系统：

# 在Dell终端上安装完整Windows + 应用
# 通用化 (移除硬件特定信息)
sysprep /generalize /oobe /shutdown

# 从WinPE启动，捕获系统分区
dism /capture-image /imagefile:D:\win11_dell_full.wim \
    /capturedir:C:\ /name:"Win11-Dell-OptiPlex-Full" \
    /compress:max

# 按层分离上传 (OS层 + 驱动层 + 应用层)
# OS层: Windows基础系统 (~8GB)
# 驱动层: Dell驱动+DNA分组 (~2GB)
# 应用层: 游戏/办公软件 (~10GB)
完整Windows镜像较大(20GB+)，Alpha阶段建议先使用WinPE镜像验证启动链
8

使用项目提供的脚本 (推荐)

项目提供了自动化镜像构建脚本 tools/mini_image/build_mini_image.ps1：

# 一键构建最小WinPE镜像 (包含bootdiag+replay)
.\tools\mini_image\build_mini_image.ps1

# 脚本自动执行:
# 1. 创建WinPE工作区
# 2. 注入bootdiag.exe
# 3. 注入boot_replay_analyzer.exe
# 4. 添加必要启动脚本
# 5. 生成WIM文件
